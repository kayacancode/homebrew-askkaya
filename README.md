# Homebrew Tap for AskKaya

This is the official Homebrew tap for [AskKaya](https://github.com/kayacancode/askkaya).

## Installation

```bash
brew tap kayacancode/askkaya
brew install askkaya
```

## Usage

```bash
askkaya login
askkaya query "How do I backup my OpenClaw setup?"
askkaya status
```
